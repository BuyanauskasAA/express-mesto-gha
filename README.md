# express-mesto-gha

`Mesto` - это веб-сервис, куда можно добавлять фотографии, удалять их и ставить лайки.

Бэкенд для проекта `Mesto`. 

## Setup

```sh
npm install
```

## Запуск проекта

- Запуск сервера

```sh
npm run start
```

- Запуск сервера с hot-reload

```sh
npm run dev
```

## Технологии

1. JavaScript
2. Express.js
3. Node.js
4. MongoDB
5. Nginx
6. API
7. JWT
